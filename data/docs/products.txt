ECOFES производит широкий ассортимент смазочных материалов:

1. Моторные масла:
   - Для бензиновых и дизельных двигателей
   - SAE 5W-30, 5W-40, 10W-40, 15W-40
   - API SN, CI-4, ACEA A3/B4

2. Трансмиссионные масла:
   - Для МКПП и редукторов
   - SAE 75W-90, 80W-90
   - API GL-4, GL-5

3. Гидравлические масла:
   - ISO VG 32, 46, 68
   - Для экскаваторов, бульдозеров, прессов

4. Индустриальные масл:
   - Компрессорные, турбинные, циркуляционные
   - Работа при высоких температурах и нагрузках
